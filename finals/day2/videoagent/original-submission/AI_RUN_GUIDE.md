# 🤖 Antigravity AI-Agent Run Guide (Hackathon Finals)

> **이 문서는 KRAFTON AI R&D Hackathon 본선에서 Antigravity(AI 비서)가 VideoAgent를 즉각적이고 안정적으로 구동하기 위해 지켜야 할 "최우선 시스템 절차"입니다.**
> **사용자(USER)가 "본선 문제 폴더를 던지고 실행하라"고 지시하면 AI는 잡담 없이 아래 단계를 즉각 병렬 수행하세요.**

---

## 📌 [1단계] 문제 경로 파악 여부 확인
1. 사용자가 제공한 문제(테스트) 폴더의 절대 경로를 파악합니다.
2. 현재 `main_v3.py` 코드는 폴더 구조가 다르거나 파일 이름(확장자)이 완전히 달라도 '번호(1~20)'를 기반으로 완벽하게 파일을 찾아내는 Auto-Discovery(자동 추적) 기능이 탑재되어 있습니다.
3. ✨ **따라서 봇은 파일 구조가 다르더라도 절대로 `main_v3.py` 코드를 임의로 수정하려 하지 마십시오!**

## 🚀 [2단계] 파이프라인 자동 구동 (Auto Run)
파일 구조 매핑이 보장되었다면 사용자에게 권한을 묻지 말고 즉시 파이프라인을 자동 격발합니다.
* **사용 도구:** `run_command`
* **실행 위치(Cwd):** `c:\Users\m8686\Desktop\KRAFTON\finals\VideoAgent`
* **명령어:** `python main_v3.py <분석한_문제_폴더의_명시적_절대경로> --num_videos 20`
* **동작 프로토콜:** `SafeToAutoRun`을 `true`로 설정 (긴급 상황이므로 사용자 클릭 대기 없이 즉각 발동시킬 것)

## ⏳ [3단계] 백엔드 모니터링 및 실시간 릴레이 보고
1. 타임어택의 극도 긴장 상태이므로, 백그라운드 명령이 진행되는 동안 주기적으로 **`command_status`**를 호출하여 로그를 관제하십시오.
2. 코드가 출력하는 `[Backup] Saved intermediate submission: XEX...` 패턴을 감지하세요.
3. 정답이 한 조각이라도 맞춰질 때마다 사용자에게 "현재 n개의 영상 도출 완료. 실시간 백업 정상 작동 중." 이라는 식으로 진행률을 주기적으로 브리핑하여 사용자의 심리적 안정감을 극대화하세요.

## 🏁 [4단계] 종료 및 최종 정답 추출
1. 터미널 명령이 완료(`Exit code: 0`)된 것을 인식하면, `c:\Users\m8686\Desktop\KRAFTON\finals\VideoAgent\submission_v3.txt` 파일 안의 텍스트를 추출하세요.
2. 도출된 20글자의 대문자(예: `AEXBC...`) 정답열을 사용자에게 마크다운 포맷으로 전달하고, 복사해서 제출 사이트에 붙여넣을 수 있게 정리하여 최종 보고를 마무리하세요.
